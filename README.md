# Revolution X - AI Trading System

نظام تداول ذكي متكامل للذهب والمعادن.

## المتطلبات
- Ubuntu 24.04
- Docker & Docker Compose
- Python 3.12+
- Node.js 20+

## البدء السريع

```bash
# 1. استنساخ المشروع
git clone https://github.com/yourusername/revolution-x.git
cd revolution-x

# 2. إعداد البيئة
cp .env.example .env
# عدل .env بإعداداتك

# 3. تشغيل
docker-compose up -d

# 4. فتح الواجهة
http://localhost:3000
